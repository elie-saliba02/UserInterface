Tutorial : 
Download main.exe on your windows machine. 
Download [LighBlue](https://punchthrough.com/lightblue/) on your phone. 
With LighBlue Create a new virtual peripheral of any type with a unique name by going in the Virtual Devices tab.  
Start the main.exe app, wait until you see your device name listed, then double click on the device to connect.
